# SQL Editor Interface

The **SQL Editor** is a powerful environment designed for executing queries, analyzing data, and managing database interactions.

## Layout Overview

The editor interface is composed of three primary panels:

1.  **Sidebar (Left)**:

    - **Connections Dropdown**: Select the active database connection.
    - **Schema Browser**: Expandable tree view of tables and columns.
    - **History & Tabs**: Toggle visibility of query history and saved tabs.

2.  **Editor Pane (Top Center)**:

    - **Monaco Editor**: A rich coding interface with syntax highlighting, line numbers, and minimap.
    - **Toolbar**: Buttons for `Execute`, `Format`, `Explain`, and `AI Assist`.
    - **Tab System**: Manage multiple query drafts simultaneously.

3.  **Results Pane (Bottom Center)**:
    - **Data Grid**: Displays query results in a responsive table.
    - **Footer**: Shows execution time, row count, and pagination controls.

## Theme Settings

The editor supports both dark and light themes, automatically adapting to your system preference or manual toggle, ensuring comfortable viewing in any lighting condition.
