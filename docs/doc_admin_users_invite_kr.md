# 사용자 초대 (User Invitation)

새로운 팀 멤버를 플랫폼에 안전하게 온보딩합니다.

## 초대 프로세스

1.  사용자 관리 페이지에서 **"사용자 초대"** 버튼을 클릭합니다.
2.  **이메일**: 사용자의 이메일 주소를 입력합니다.
3.  **역할**: 초기 역할을 할당합니다 (예: User, Admin).
4.  **전송**: 시스템이 보안 처리된 시간 제한이 있는 등록 링크가 포함된 이메일을 발송합니다.

## 초대 상태

- **대기 중 (Pending)**: 사용자가 목록에 "초대됨(Invited)" 상태 배지와 함께 표시됩니다.
- **만료됨 (Expired)**: 링크가 만료된 경우, **초대 재전송** 작업을 사용하여 새 링크를 생성하세요.
- **수락됨 (Accepted)**: 사용자가 설정을 완료하면 상태가 "활성(Active)"으로 변경됩니다.
