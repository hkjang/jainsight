# SQL 에디터 인터페이스

**SQL 에디터**는 쿼리 실행, 데이터 분석, 그리고 데이터베이스 상호작용 관리를 위해 설계된 강력한 환경입니다.

## 레이아웃 개요

에디터 인터페이스는 세 개의 주요 패널로 구성됩니다:

1.  **사이드바 (왼쪽)**:

    - **연결 드롭다운**: 활성 데이터베이스 연결을 선택합니다.
    - **스키마 브라우저**: 테이블과 컬럼을 트리 구조로 펼쳐볼 수 있습니다.
    - **히스토리 및 탭**: 쿼리 기록과 저장된 탭의 가시성을 전환합니다.

2.  **에디터 창 (상단 중앙)**:

    - **Monaco 에디터**: 구문 강조, 줄 번호, 미니맵을 지원하는 풍부한 코딩 인터페이스입니다.
    - **툴바**: `실행(Execute)`, `포맷(Format)`, `실행 계획(Explain)`, `AI 지원(AI Assist)` 버튼이 있습니다.
    - **탭 시스템**: 여러 쿼리 초안을 동시에 관리할 수 있습니다.

3.  **결과 창 (하단 중앙)**:
    - **데이터 그리드**: 쿼리 결과를 반응형 테이블로 표시합니다.
    - **푸터(Footer)**: 실행 시간, 행 수, 페이지네이션 컨트롤을 보여줍니다.

## 테마 설정

에디터는 다크 모드와 라이트 모드를 모두 지원하며, 시스템 설정에 자동으로 적응하거나 수동으로 전환하여 어떤 조명 환경에서도 편안한 가독성을 보장합니다.
