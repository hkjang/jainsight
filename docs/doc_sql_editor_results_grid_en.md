# Result Management

The **Results Grid** is where your data comes to life. It handles large datasets efficiently and provides tools for deeper analysis.

## Navigation & Sorting

- **Pagination**: Currently displays up to 25 rows per page. Use the jump input to skip to a specific page.

- **Keyboard Navigation**: Use arrow keys to move the selection focus between cells.

## Advanced Actions

- **Context Menu**: Right-click any cell or row to access actions:
  - **Copy Cell**: Copy the specific value.
  - **Copy Row**: Copy the entire row as JSON.
  - **Filter by Value**: Automatically add a `WHERE` clause for the selected value.
- **Pin Results**: Save the current result set to a "pinned" state, allowing you to run new queries without losing these specific results.
- **Export**:
  - **CSV**: Download all rows as a standard CSV file.
  - **JSON**: Download or copy results in JSON format.
