# 키보드 단축키

SQL 에디터의 필수 키보드 단축키를 익혀 작업을 가속화하세요.

| 동작 (Action)     | Windows/Linux          | Mac                   |
| :---------------- | :--------------------- | :-------------------- |
| **쿼리 실행**     | `Ctrl` + `Enter`       | `Cmd` + `Enter`       |
| **SQL 포맷팅**    | `Ctrl` + `Shift` + `F` | `Cmd` + `Shift` + `F` |
| **찾기**          | `Ctrl` + `F`           | `Cmd` + `F`           |
| **바꾸기**        | `Ctrl` + `H`           | `Cmd` + `H`           |
| **주석 처리**     | `Ctrl` + `/`           | `Cmd` + `/`           |
| **새 탭**         | `Alt` + `T`            | `Option` + `T`        |
| **탭 닫기**       | `Alt` + `W`            | `Option` + `W`        |
| **다음 탭**       | `Alt` + `→`            | `Option` + `→`        |
| **이전 탭**       | `Alt` + `←`            | `Option` + `←`        |
| **사이드바 토글** | `Ctrl` + `B`           | `Cmd` + `B`           |
