# 대시보드 통계 및 위젯

통계 그리드는 작업 공간의 사용량과 안정성에 대한 정량적인 개요를 제공합니다.

## 지표 카드 (Metric Cards)

### 1. 전체 연결 (Total Connections)

- **아이콘**: 🔗
- **설명**: 현재 작업 공간에 설정된 총 데이터베이스 연결 수를 표시합니다.
- **목적**: 데이터베이스 통합 규모를 파악하는 데 도움을 줍니다.

### 2. 실행된 쿼리 (Executed Queries)

- **아이콘**: 📊
- **설명**: 실행된 총 쿼리 수입니다.
- **시각적 요소**:
  - **미니 스파크라인(Sparkline)**: 최근 7개 구간의 쿼리량 추세를 보여주는 작은 선 그래프입니다.
  - **성공률 바(Success Rate Bar)**: 전체 쿼리 대비 성공한 쿼리의 비율을 막대 그래프로 보여줍니다. 성공률이 높으면(>80%) 초록색, 중간이면(>50%) 주황색, 낮으면 빨간색으로 표시됩니다.

### 3. 실패한 쿼리 (Failed Queries)

- **아이콘**: ⚠️
- **설명**: 에러가 발생한 쿼리의 수입니다.
- **목적**: 이 수치를 모니터링하여 연결 문제나 빈번한 SQL 오류를 식별할 수 있습니다.

## 인터랙션

- 카드 위에 마우스를 올리면 "글래스모피즘(Glassmorphism)" 효과가 적용되며 카드가 살짝 떠올라 활성 UI 요소임을 나타냅니다.
- 대시보드 로드 시 숫자가 0부터 카운트업되는 애니메이션이 적용되어 역동적인 느낌을 줍니다.
