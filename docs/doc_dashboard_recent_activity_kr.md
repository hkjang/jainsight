# 최근 활동 (Recent Activity)

**최근 활동** 섹션은 시스템 내에서 수행된 최신 작업 로그를 실시간으로 제공하여, 쿼리 사용량과 시스템 상호작용을 엄격하게 모니터링할 수 있도록 돕습니다.

## 활동 테이블 컬럼

테이블은 각 활동에 대해 다음 정보를 표시합니다:

1.  **시간 (Time)**: 실행 후 경과된 상대 시간(예: "5분 전"). 마우스를 올리면 정확한 날짜와 시간을 확인할 수 있습니다.
2.  **사용자 (User)**: 쿼리를 실행한 사용자의 이름입니다.
3.  **연결 (Connection)**: 쿼리에 사용된 특정 데이터베이스 연결입니다.
4.  **쿼리 (Query)**: 실행된 SQL 쿼리의 일부입니다. 긴 쿼리는 가독성을 위해 잘려서 표시됩니다.
5.  **상태 (Status)**: 쿼리 실행 결과입니다.
    - **성공 (✓)**: 오류 없이 실행됨.
    - **실패 (✗)**: 실행 중 오류 발생.
6.  **액션 (Action)**:
    - **복사 (📋)**: 전체 SQL 쿼리를 클립보드로 빠르게 복사하여 재사용하거나 디버깅할 수 있는 버튼입니다.

## 빈 상태 (Empty State)

최근 실행된 쿼리가 없을 경우, "비어 있음" 일러스트와 함께 SQL 에디터에서 첫 쿼리를 시작하라는 안내가 표시됩니다.
