# 대시보드 개요

**대시보드(Dashboard)**는 Jainsight Enterprise DB Hub의 중앙 제어 센터 역할을 합니다. 시스템 상태, 데이터베이스 연결 현황, 그리고 최근 쿼리 활동에 대한 즉각적인 가시성을 제공합니다.

## 주요 구성 요소

대시보드는 크게 네 가지 영역으로 구분됩니다:

1.  **헤더 및 상태**: 사용자 인사말, 시스템 상태, 그리고 주요 탐색 버튼.
2.  **통계 그리드(Statistics Grid)**: 연결 및 쿼리 성능에 대한 실시간 지표.
3.  **최근 활동(Recent Activity)**: 플랫폼 전체에서 실행된 최신 쿼리 피드.
4.  **빠른 작업(Quick Actions)**: 자주 사용하는 도구로 이동하는 단축키 지원 버튼.

## 대시보드 접속 방법

언제든지 다음 방법으로 대시보드로 돌아올 수 있습니다:

- 사이드바 왼쪽 상단의 **Jainsight** 로고 클릭.
- "작업 공간" 메뉴 그룹에서 **대시보드** 클릭.
- 루트 URL `/`로 이동.

## 헤더 영역 (Header Area)

헤더는 시간대별 인사말(예: "좋은 아침이에요")과 사용자 이름을 표시하여 사용자를 환영합니다.

- **시스템 상태**: 초록색으로 펄럭이는 표시기는 시스템이 정상 작동 중임을 나타냅니다.
- **주요 작업**:
  - **+ 새 연결**: 새로운 데이터베이스 연결 생성 페이지로 빠르게 이동합니다.
  - **⚡ SQL 에디터**: SQL 쿼리 에디터 인터페이스로 즉시 이동합니다.
