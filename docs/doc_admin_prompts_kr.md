# 프롬프트 템플릿 (Prompt Templates)

NL2SQL 변환 및 기타 AI 작업에 사용되는 시스템 프롬프트를 관리하고 버전을 제어합니다.

## 템플릿 관리

- **버전 관리**: 버전 번호(v1, v2...)로 프롬프트 변경 사항을 추적합니다.
- **승인 워크플로우**: 프롬프트는 프로덕션에서 사용되기 전에 승인("대기" vs "승인됨")이 필요할 수 있습니다.
- **변수**: 지능형 변수 감지. 실행 중에 동적으로 대체되는 `{{variable}}` 구문(예: `{{schema}}`, `{{userQuery}}`)을 사용합니다.

## 에디터 기능

- **미리보기**: 변수에 대한 구문 강조 표시가 있는 확장 가능한 콘텐츠 뷰.
- **유효성 검사**: 사용된 변수의 실시간 감지.
- **테스트 렌더러**: `schema`와 `userQuery`에 대한 모의 데이터를 제공하여 프롬프트 렌더링을 시뮬레이션하고 AI에 전송될 정확한 내용을 확인합니다.

## 용도 태그

특정 사용 사례별로 프롬프트를 분류합니다:

- **NL2SQL**: 핵심 자연어 SQL 변환.
- **설명 (Explain)**: SQL 설명 생성.
- **최적화 (Optimize)**: 쿼리 최적화 제안.
- **검증 (Validate)**: 보안 및 로직 검증 프롬프트.
