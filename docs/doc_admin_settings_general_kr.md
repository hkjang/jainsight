# 일반 설정 (General Settings)

전역 플랫폼 동작 및 정책을 구성합니다.

## 구성 카테고리

### 1. 일반 및 보안 (General & Security)

- **사이트 ID**: 사이트 이름과 설명을 설정합니다.
- **유지보수**: 유지보수 모드를 전환하여 관리자 외 사용자의 접근을 차단합니다.
- **보안 정책**:
  - **세션 타임아웃**: 자동 로그아웃 시간.
  - **MFA**: 다단계 인증(Multi-Factor Authentication) 강제.
  - **비밀번호 정책**: 최소 길이 요건 설정.

### 2. 쿼리 실행 (Query Execution)

- **기본 Limit**: 사용자가 LIMIT를 지정하지 않았을 때 반환되는 최대 행 수.
- **최대 실행 시간**: DB 락업 방지를 위한 장기 실행 쿼리 타임아웃.
- **DDL 허용**: 위험한 DDL 명령을 허용/거부하는 전역 스위치.
- **감사 정책**: 모든 쿼리 로깅 vs 위험한 쿼리만 로깅 선택.

### 3. API 및 알림 (API & Notifications)

- **속도 제한**: 전역 기본 API 속도 제한.
- **알림 (Notifications)**:
  - **이메일**: 시스템 이메일 활성화.
  - **Slack**: 슬랙 연동을 위한 웹훅 URL 구성.
  - **알림 임계값**: 알림을 트리거할 위험도 점수(0-100) 설정.
