# 히스토리 및 저장된 쿼리

통합된 히스토리 및 쿼리 저장 관리 시스템으로 작업 내용을 안전하게 보관하세요.

## 쿼리 히스토리 (Query History)

실행된 모든 쿼리는 왼쪽 사이드바의 **History** 탭에 자동으로 기록됩니다.

- **검색**: 쿼리 키워드로 히스토리를 필터링할 수 있습니다.
- **복원**: 히스토리 항목을 클릭하여 즉시 에디터로 불러올 수 있습니다.
- **보존**: 히스토리는 로컬에 저장되어 세션이 종료되어도 최근 작업 내용이 유지됩니다.

## 저장된 쿼리 (Saved Queries)

자주 사용하거나 복잡한 쿼리를 저장하여 빠르게 접근하세요.

- **생성**: 툴바의 "저장(Save)" 아이콘을 클릭합니다. 이름과 설명을 지정할 수 있습니다.
- **공유**: 저장된 쿼리를 **공개(Public)**로 설정하면 조직 내 다른 사용자도 접근할 수 있습니다 (관리자 전용).
- **관리**: 사이드바에서 저장된 쿼리의 이름을 변경하거나 삭제할 수 있습니다.

## 탭 (Tabs)

탭을 사용하여 여러 작업을 동시에 수행하세요. 탭을 더블 클릭하여 이름(예: "사용자 분석", "수익 확인")을 변경하면 더 체계적으로 관리할 수 있습니다.
