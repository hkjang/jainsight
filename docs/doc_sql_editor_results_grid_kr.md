# 결과 관리 (Result Management)

**결과 그리드(Results Grid)**는 데이터가 살아 숨 쉬는 곳입니다. 대규모 데이터셋을 효율적으로 처리하고 심층 분석을 위한 도구를 제공합니다.

## 탐색 및 정렬 (Navigation & Sorting)

- **페이지네이션**: 현재 페이지당 최대 25개 행을 표시합니다. 점프 입력창을 사용하여 특정 페이지로 건너뛸 수 있습니다.
- **정렬**: 컬럼 헤더를 클릭하여 오름차순(🔼), 내림차순(🔽), 기본 순서 간에 전환할 수 있습니다.
- **키보드 탐색**: 화살표 키를 사용하여 셀 간 선택 포커스를 이동할 수 있습니다.

## 고급 작업 (Advanced Actions)

- **컨텍스트 메뉴**: 아무 셀이나 행을 우클릭하여 다음 작업에 접근할 수 있습니다:
  - **셀 복사**: 특정 값을 복사합니다.
  - **행 복사**: 전체 행을 JSON으로 복사합니다.
  - **값으로 필터링**: 선택한 값에 대한 `WHERE` 절을 자동으로 추가합니다.
- **결과 고정 (Pin Results)**: 현재 결과 세트를 "고정" 상태로 저장하여, 이 특정 결과를 잃지 않고 새로운 쿼리를 실행할 수 있게 합니다.
- **내보내기 (Export)**:
  - **CSV**: 모든 행을 표준 CSV 파일로 다운로드합니다.
  - **JSON**: 결과를 JSON 형식으로 다운로드하거나 복사합니다.
