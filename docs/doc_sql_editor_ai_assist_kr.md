# AI 지원 (AI Assistance)

생성형 AI의 힘을 활용하여 쿼리를 더 빠르게 작성하고 데이터를 손쉽게 시각화하세요.

## Text-to-SQL

복잡한 SQL을 직접 작성하는 대신, 자연어로 의도를 입력하기만 하면 됩니다.

1. 툴바에서 **AI Assist** 버튼을 클릭합니다.
2. 프롬프트를 입력합니다: _"지난달 주문 건수 기준 상위 5명의 사용자를 보여줘."_
3. AI가 생성한 SQL을 확인하고 에디터에 즉시 삽입할 수 있습니다.

## 스마트 설명 (Smart Explanations)

복잡한 쿼리를 이해하기 어려우신가요? 쿼리를 드래그한 후 우클릭 메뉴나 툴바를 사용하여 AI에게 "설명해줘(Explain this)"라고 요청하세요.

## 데이터 시각화

에디터는 결과 데이터를 기반으로 차트를 자동 생성할 수 있습니다.

- **파이 차트**: 범주형 분포(예: 사용자 상태)를 확인하는 데 적합합니다.
- **바 차트**: 수치 비교(예: 지역별 매출)에 이상적입니다.
- **자동 감지**: 시스템이 결과 컬럼을 분석하여 가장 적합한 차트 유형을 자동으로 제안합니다.
