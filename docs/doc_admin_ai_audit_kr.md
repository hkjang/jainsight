# AI 감사 로그 (AI Audit Logs)

시스템과 AI 모델 간의 모든 상호 작용에 대한 포괄적이고 세분화된 로그입니다.

## 로그 상세 (Log Details)

각 감사 항목은 다음을 포함하는 상세 기록을 생성합니다:

- **타임스탬프**: 요청의 정확한 시간.
- **상태**: 성공, 실패 또는 차단됨(정책에 의해).
- **입력 및 출력**: 정확한 사용자 입력 텍스트 및 생성된 SQL.
- **메타데이터**: 사용된 공급자, 사용된 모델, 토큰 수(입력/출력) 및 지연 시간.
- **오류**: 구체적인 오류 메시지 또는 차단 사유(예: "DDL 차단됨").

## 필터링 및 탐색

- **검색**: 상태(성공/실패), 날짜 범위 또는 공급자별로 로그를 필터링합니다.
- **페이지네이션**: 대량의 이력을 효율적으로 탐색할 수 있습니다.
- **키보드 지원**: 표준 단축키(화살표 키, J/K)를 사용하여 로그를 탐색하고 Enter를 눌러 상세 정보를 봅니다.

## 내보내기 (Export)

- **CSV/JSON**: 외부 분석 또는 규정 준수 보고를 위해 전체 감사 이력 또는 필터링된 결과를 다운로드합니다.
