# User Invitation

Securely onboard new team members to the platform.

## Invitation Process

1.  Click the **"Invite User"** button on the User Management page.
2.  **Email**: Enter the user's email address.
3.  **Role**: Assign an initial role (e.g., User, Admin).
4.  **Send**: The system dispatches an email with a secure, time-limited registration link.

## Invitation Status

- **Pending**: The user appears in the list with an "Invited" status badge.
- **Expired**: If the link expires, use the **Resend Invite** action to generate a new one.
- **Accepted**: Once the user completes setup, their status changes to "Active".
