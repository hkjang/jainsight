# IP 보안 (IP Security)

Jainsight 플랫폼에 대한 네트워크 수준의 접근 제어를 관리합니다.

## 전역 IP 허용 목록 (Global IP Allowlist)

요청자의 IP 주소를 기반으로 전체 관리자 콘솔 또는 특정 API 엔드포인트에 대한 접근을 제한합니다.

- **CIDR 지원**: 표준 CIDR 표기법(예: `192.168.1.0/24`)을 사용하여 범위를 정의합니다.
- **설명**: 각 IP 항목에 식별 가능한 이름(예: "서울 오피스 VPN")을 태그합니다.
- **강제 적용 범위**:
  - **관리자 콘솔**: 웹 인터페이스 로그인을 제한합니다.
  - **API**: 프로그래밍 방식 접근을 제한합니다 (특정 API 키 설정으로 재정의 가능).

## 사고 대응 (Incident Response)

- **차단 목록**: 의심스러운 활동을 보이는 특정 IP를 일시적으로 차단합니다 (속도 제한 규칙에 의해 자동 채워짐).
- **접근 로그**: IP 제한으로 인한 "접근 거부" 이벤트의 필터링된 스트림을 확인합니다.
