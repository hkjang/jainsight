# 연결 생성 및 수정 (Creating & Editing Connections)

외부 데이터베이스 시스템에 대한 액세스를 안전하게 구성합니다.

## 지원되는 데이터베이스

Jainsight는 현재 다음 데이터베이스 유형을 지원합니다:

- **PostgreSQL**: (기본 포트 5432)
- **MySQL**: (기본 포트 3306)
- **MariaDB**: (기본 포트 3306)
- **Oracle**: (기본 포트 1521)
- **MSSQL**: (기본 포트 1433)
- **SQLite**: (파일 경로 기반)

## 구성 필드

- **연결 이름 (Connection Name)**: 이 연결을 식별할 수 있는 별칭 (예: "Production Core").
- **유형 (Type)**: 데이터베이스 벤더를 선택합니다.
- **호스트 (Host)**: IP 주소 또는 도메인 이름.
- **포트 (Port)**: 리스너 포트 번호.
- **데이터베이스 (Database)**: 접속할 특정 데이터베이스 이름.
- **사용자 이름 (Username)**: 데이터베이스 사용자 자격 증명.
- **비밀번호 (Password)**: 비밀번호 (안전하게 저장됨).
- **연결 테스트**: 저장하기 전에 항상 이 버튼을 클릭하여 자격 증명이 유효하고 방화벽이 액세스를 허용하는지 확인하세요.
