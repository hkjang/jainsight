# 연결 목록 (Connections List)

**연결 목록**은 데이터베이스를 위한 주소록 역할을 하며, 연결된 모든 데이터 소스에 대한 상태 인식 개요를 제공합니다.

## 대시보드 통계

페이지 상단의 4개 상태 카드는 즉각적인 상태 점검 정보를 제공합니다:

- **전체 연결**: 구성된 총 데이터베이스 수.
- **온라인**: 현재 활성 상태이며 접근 가능한 연결 수.
- **오프라인**: 마지막 상태 점검에서 실패한 연결 수.
- **타입 수**: 사용 중인 데이터베이스 시스템(예: Postgres, MySQL)의 다양성.

## 연결 카드

각 연결은 다음 정보를 포함하는 카드로 표시됩니다:

1.  **상태 표시기**: 깜박이는 점(초록/온라인, 빨강/오프라인, 주황/테스트 중)으로 실시간 상태 피드백을 제공합니다.
2.  **데이터베이스 아이콘**: DB 유형을 나타내는 시각적 식별자(예: PostgreSQL의 코끼리).
3.  **세부 정보**: 호스트 IP, 포트 및 데이터베이스 이름.
4.  **생성일**: 연결이 추가된 날짜.

## 작업 (Actions)

- **테스트 (🔌)**: 연결 상태를 확인하기 위해 핑(Ping)을 전송합니다.
- **쿼리 (⚡)**: 해당 연결이 미리 선택된 상태로 SQL 에디터를 엽니다.
- **스키마 (📊)**: 해당 연결에 대한 스키마 탐색기를 엽니다.
- **복제 (📋)**: 연결 설정을 복사합니다(개발/운영 등 유사 환경 설정 시 유용).
- **수정 (✏️)**: 연결 정보를 수정합니다.
- **삭제 (🗑️)**: 연결 구성을 제거합니다.
