# 사용자 감사 로그 (User Audit Logs)

시스템 전반의 AI 감사와는 별개로, 플랫폼 내에서 실행된 SQL 쿼리에 대한 사용자용 이력입니다.

## 기능 (Features)

- **쿼리 이력**: 실행된 모든 SQL 쿼리의 시간순 목록을 봅니다.
- **실행 상세 정보**:
  - **사용자**: 쿼리를 실행한 사람.
  - **연결**: 사용된 데이터베이스 연결.
  - **소요 시간**: 실행 시간(밀리초 단위, 성능 인식을 위한 색상 구분).
  - **상태**: 성공 또는 실패 상태.
  - **상대적 시간**: 빠른 상황 파악을 위한 "방금 전", "5분 전" 표시.

## 작업 (Actions)

- **검색**: 쿼리 내용, 사용자 이름 또는 연결 이름으로 실시간 검색.
- **쿼리 복사**: 원본 SQL 쿼리를 클립보드로 복사하여 재사용할 수 있는 원클릭 버튼.
- **뷰 확장**: 잘린 쿼리를 클릭하여 전체 SQL 텍스트를 확인합니다.
- **자동 새로고침**: 5초마다 새로운 쿼리를 자동으로 확인하는 토글 기능.
