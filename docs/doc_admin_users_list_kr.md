# 사용자 관리 (User Management)

**사용자 관리** 페이지는 Jainsight 플랫폼 내의 모든 사용자 계정을 관리하는 중앙 허브입니다.

## 사용자 목록 (User List)

메인 테이블에는 각 사용자에 대한 포괄적인 세부 정보가 표시됩니다:

- **사용자 프로필**: 이름 및 이메일 주소.
- **역할**: 현재 시스템 역할 (예: Admin, User).
- **상태 (Status)**:
  - **활성 (Active)** (초록색): 정상 계정 상태.
  - **잠김 (Locked)** (빨간색): 보안상의 이유 또는 관리자 조치로 인해 차단된 계정.
  - **초대됨 (Invited)** (노란색): 초대는 발송되었으나 아직 수락되지 않음.
- **소스**: 계정 생성 방식 (예: Email, SSO).
- **마지막 로그인**: 마지막으로 성공한 로그인 타임스탬프.

## 작업 (Actions)

### 일괄 작업 (Bulk Actions)

체크박스를 사용하여 여러 사용자를 선택하고 일괄 작업을 수행할 수 있습니다:

- **활성화 (Activate)**: 잠긴 사용자의 액세스 권한을 복구합니다.
- **잠금 (Lock)**: 선택한 사용자의 액세스를 일시적으로 비활성화합니다.
- **삭제 (Delete)**: 사용자 계정을 영구적으로 제거합니다 (되돌릴 수 없음).

### 개별 작업

사용자 행의 "추가 옵션" (점 3개) 메뉴를 클릭합니다:

- **잠금 해제 (Unlock)**: 잠긴 계정을 복구하는 특정 작업.
- **초대 재전송**: '초대됨' 상태인 사용자에게 이메일을 다시 보냅니다.
- **강제 로그아웃**: 사용자의 현재 세션을 즉시 무효화합니다.

## 검색 및 필터

- **검색**: 이름 또는 이메일로 필터링합니다.
- **상태 필터**: 활성, 잠김, 또는 초대된 사용자만 봅니다.
