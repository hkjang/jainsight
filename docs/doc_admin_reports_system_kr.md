# 시스템 리포트 (System Reports)

사용 패턴, 보안 위험, 운영 상태에 대한 분석 인사이트를 확보합니다.

## 대시보드 개요

실시간 지표 시각화:

- **전체 및 활성 사용자**: 사용자 참여 통계.
- **쿼리 볼륨**: 전체 쿼리 대 차단된 쿼리.
- **위험도 점수**: 시스템 전반의 평균 위험 수준.

## 리포트 탭

1.  **개요 (Overview)**: 시간 경과(7/30/90일)에 따른 쿼리 양과 차단 시도를 보여주는 추세 차트. 그룹/팀별 사용량 분석.
2.  **활동 분석 (Activity Analysis)**: 쿼리 수를 기준으로 상위 사용자를 순위 매김하며, 개별 활동 추세를 미니 차트로 표시합니다.
3.  **권한 분석 (Permissions Analysis)**: 거의 사용하지 않는 역할을 가진 "과잉 권한" 사용자를 식별하여 보안 최적화를 제안합니다.
4.  **위험 분석 (Risk Analysis)**: 차단된 DROP 테이블이나 고위험 비인가 접근 시도와 같은 보안 사고의 우선순위 피드.

## 리포트 내보내기

- **JSON 리포트**: 모든 리포트 데이터의 전체 스냅샷.
- **CSV 내보내기**: 사용자 활동 및 권한 문제 테이블에 대한 별도 내보내기.
