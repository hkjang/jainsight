# System Reports

Gain analytics insights into usage patterns, security risks, and operational health.

## Dashboard Overview

Real-time metrics visualizing:

- **Total & Active Users**: User engagement stats.
- **Query Volume**: Total vs Blocked queries.
- **Risk Score**: Average system-wide risk level.

## Report Tabs

1.  **Overview**: Trends chart showing query volume and blocked attempts over time (7/30/90 days). Breakdown of usage by Group/Team.
2.  **Activity Analysis**: Top users ranked by query count, with mini-charts showing their individual activity trends.
3.  **Permissions Analysis**: Identifies "Over-privileged" users who have roles they rarely use, suggesting security optimizations.
4.  **Risk Analysis**: High-priority feed of security incidents, such as blocked DROP tables or high-risk unauthorized access attempts.

## Exporting Reports

- **JSON Report**: Full snapshot of all report data.
- **CSV Exports**: Specific exports for User Activity and Permission Issues tables.
