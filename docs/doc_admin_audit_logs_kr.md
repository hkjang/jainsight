# 감사 로그 (Audit Logs)

모든 시스템 활동 및 보안 이벤트에 대한 포괄적이고 변경 불가능한 기록입니다.

## 로그 인터페이스

- **타임라인**: 누가, 언제, 무엇을 했는지, 그리고 그 결과를 시간순으로 보여주는 피드입니다.
- **아이콘 및 색상**: 작업은 시각적으로 우선순위가 지정됩니다 (성공은 초록색, 실패/차단은 빨간색).

## 필터링

로그를 좁혀 특정 이벤트를 찾습니다:

- **검색**: 사용자 이름, 작업 유형, 또는 리소스 이름으로 필터링합니다.
- **리소스 필터**: `Query`, `User`, `Role`, `System` 등 특정 유형으로 제한합니다.
- **상태 필터**: 실패나 경고만 표시하여 문제를 빠르게 파악합니다.
- **시간 범위**: 오늘, 7일, 30일, 전체 기간으로 전환합니다.

## 로그 상세

로그 항목을 클릭하면 전체 실행된 SQL 쿼리, 차단 사유, IP 주소 등 이벤트의 구체적인 세부 정보가 포함된 JSON 페이로드를 확인할 수 있습니다.

## 내보내기 (Export)

현재 필터링된 로그 세트를 CSV 파일로 다운로드하여 외부 규정 준수 보고나 오프라인 분석에 사용할 수 있습니다.
