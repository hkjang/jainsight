# 시스템 API 키 (System API Keys)

Jainsight 플랫폼에 대한 프로그래매틱한 접근을 관리합니다.

## 키 관리

- **API 키 목록**: 모든 활성 키, 소유자, 현재 상태를 조회합니다.
- **접두사 (Prefix)**: 키는 식별을 용이하게 하기 위해 접두사(예: `jsk_live_***`)와 함께 표시되며, 비밀 값은 안전하게 유지됩니다.
- **상태**: 키는 활성, 취소됨, 또는 만료됨 상태일 수 있습니다.

## 키 생성

새 API 키를 생성할 때 다음을 제어할 수 있습니다:

- **스코프 (Scopes)**: 정확한 권한을 정의합니다 (예: `query:read`, `query:execute`, `schema:read`).
- **속도 제한 (Rate Limit)**: 남용 방지를 위해 분당 요청 수를 제한합니다.
- **만료 기간**: 유효 기간을 설정합니다 (예: 30일, 1년, 또는 무제한).
- **IP 화이트리스트**: 보안 강화를 위해 특정 IP 주소 대역에서의 사용만 허용합니다.

## 사용량 모니터링

키의 "상세" 버튼을 클릭하여 확인합니다:

- **총 사용량**: 전체 기간 동안의 요청 수.
- **추이**: 최근 7일간의 사용 활동을 보여주는 막대 차트.
