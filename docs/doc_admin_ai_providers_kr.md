# AI 공급자 (AI Providers)

플랫폼 전반에서 사용되는 다양한 AI 서비스(LLM) 로의 연결을 관리합니다.

## 지원되는 공급자 유형

- **vLLM**: 고성능 오픈소스 모델 서빙.
- **Ollama**: 프라이버시와 비용 절감을 위한 로컬 LLM 실행.
- **OpenAI**: 표준 GPT-4/3.5 연동.

## 공급자 구성

- **엔드포인트**: API URL (예: Ollama의 경우 `http://localhost:11434`).
- **API 키**: 안전하게 저장되는 자격 증명 (로컬 공급자의 경우 선택 사항).
- **우선순위 (Priority)**: 폴백 순서를 정의합니다. 높은 우선순위 공급자가 먼저 시도됩니다.
- **타임아웃 및 재시도**: 네트워크 안정성을 위한 회복 탄력성 설정.

## 테스트 및 검증

- **연결 테스트**: "전체 테스트" 버튼으로 모든 활성 공급자에 대한 연결을 즉시 검증합니다.
- **지연 시간 확인**: 성능 보장을 위해 응답 시간을 밀리초 단위로 측정합니다.
- **일괄 관리**: 유지보수를 위해 여러 공급자를 한 번에 활성화/비활성화할 수 있습니다.
