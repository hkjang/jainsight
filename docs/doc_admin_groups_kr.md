# 그룹 관리 (Group Management)

권한 관리와 협업을 용이하게 하기 위해 사용자를 논리적인 구조로 조직합니다.

## 그룹 계층 구조 (Group Hierarchy)

왼쪽 패널은 세 가지 레벨을 사용하여 조직 및 팀의 트리 뷰를 표시합니다:

1.  **조직 (Organization - 건물 아이콘)**: 최상위 부서.
2.  **프로젝트 (Project - 폴더 아이콘)**: 중간 레벨 그룹.
3.  **업무 (Task - 클립보드 아이콘)**: 특정 기능 단위.

## 그룹 생성

- **수동 그룹**: 이름, 유형, 상위 그룹을 지정하여 생성합니다.
- **자동 그룹 (Auto Group)**: JSON 조건(예: `{"emailDomain": "@dev.com"}`)을 정의하여, 등록 또는 업데이트 시 조건에 맞는 사용자를 자동으로 이 그룹에 추가합니다.

## 멤버 (Members)

멤버십을 관리할 그룹을 선택하세요.

- **조회**: 그룹 내 사용자 목록을 확인합니다.
- **제거**: 특정 그룹에서 사용자를 제거합니다.
- **통계**: 그룹 유형별(조직, 프로젝트, 업무) 총 멤버 수를 한눈에 파악할 수 있습니다.
