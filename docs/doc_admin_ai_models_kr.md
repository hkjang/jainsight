# AI 모델 (AI Models)

다양한 시스템 용도에 맞게 구체적인 모델 구성을 정의합니다.

## 모델 구성

- **기본 모델 ID**: 공급자 측의 실제 모델 이름 (예: `gpt-4`, `llama3`).
- **공급자 연결**: 구성된 AI 공급자와 모델을 연결합니다.
- **매개변수**: `Temperature` (창의성), `Top-P`, `Max Tokens`를 미세 조정합니다.
- **시스템 프롬프트**: 이 특정 모델 구성에 대한 기본 시스템 동작을 설정합니다.

## 용도 유형 (Purpose Types)

성능과 비용 최적화를 위해 모델을 특정 작업에 할당합니다:

- **SQL 생성**: 코드 생성에 최적화된 모델 (예: 엄격하고 낮은 temperature).
- **설명 (Explanation)**: 쿼리 설명을 위한 자연어 추론에 뛰어난 모델.
- **일반 (General)**: 기타 작업에 대한 예비 모델.

## 테스트

- **인터랙티브 테스트**: UI에서 직접 샘플 "SELECT 1" 프롬프트를 전송하여 모델이 올바르게 응답하는지 확인하고 지연 시간을 체크합니다.
