# 스키마 상세 (Schema Details)

**스키마 상세** 패널(오른쪽 패널)은 선택된 테이블이나 뷰에 대한 심층 정보를 보여줍니다.

## 헤더 정보

- **제목**: 테이블 이름과 함께 유형(TABLE/VIEW)을 구별되는 배지로 표시합니다.
- **메타데이터**: 총 컬럼 수, 기본 키(PK) 수, Null 허용 컬럼 수를 보여줍니다.
- **작업 (Actions)**:
  - **SELECT 생성**: `SELECT * FROM table LIMIT 100` 쿼리를 클립보드에 복사합니다.
  - **에디터에서 열기**: 해당 테이블에 대한 쿼리가 미리 작성된 상태로 SQL 에디터로 이동합니다.

## 컬럼 그리드

선택된 객체의 모든 컬럼을 나열하는 상세 테이블입니다.

- **PK**: 기본 키(Primary Key) 컬럼을 나타내는 열쇠 아이콘.
- **컬럼명**: 컬럼의 기술적 이름 (고정폭 글꼴).
- **데이터 타입**: 구체적인 데이터베이스 타입 (예: `VARCHAR`, `INTEGER`).
- **Nullable**: `NULL`은 노란색 배지, `NOT NULL`은 빨간색 배지로 표시.
- **복사 작업**: 컬럼 이름을 클립보드로 복사하는 원클릭 버튼.

## 일괄 작업

- **컬럼 검색**: 테이블 내 컬럼을 필터링합니다.
- **전체 컬럼 복사**: 쉼표로 구분된 전체 컬럼 이름 목록을 복사합니다. SQL `INSERT`나 `SELECT` 문을 작성할 때 유용합니다.
