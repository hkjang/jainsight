# Keyboard Shortcuts

Become a power user with these essential keyboard shortcuts for the SQL Editor.

| Action             | Shortcut (Win/Linux)   | Shortcut (Mac)        |
| :----------------- | :--------------------- | :-------------------- |
| **Execute Query**  | `Ctrl` + `Enter`       | `Cmd` + `Enter`       |
| **Format SQL**     | `Ctrl` + `Shift` + `F` | `Cmd` + `Shift` + `F` |
| **Find**           | `Ctrl` + `F`           | `Cmd` + `F`           |
| **Replace**        | `Ctrl` + `H`           | `Cmd` + `H`           |
| **Comment Line**   | `Ctrl` + `/`           | `Cmd` + `/`           |
| **New Tab**        | `Alt` + `T`            | `Option` + `T`        |
| **Close Tab**      | `Alt` + `W`            | `Option` + `W`        |
| **Next Tab**       | `Alt` + `Right`        | `Option` + `Right`    |
| **Previous Tab**   | `Alt` + `Left`         | `Option` + `Left`     |
| **Toggle Sidebar** | `Ctrl` + `B`           | `Cmd` + `B`           |
