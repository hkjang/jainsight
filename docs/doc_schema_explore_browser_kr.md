# 스키마 브라우저 (Schema Browser)

**스키마 브라우저**는 데이터베이스 구조에 대한 계층적 뷰를 제공하여, 쿼리를 작성하지 않고도 테이블, 뷰, 컬럼을 탐색할 수 있게 해줍니다.

## 인터페이스 구성 요소

### 1. 연결 선택기 (Connection Selector)

- **목적**: 탐색할 데이터베이스 연결을 선택합니다.
- **동작**: 드롭다운 메뉴에서 연결을 선택합니다. 연결이 없는 경우 "연결 추가" 버튼이 제공됩니다.

### 2. 테이블 목록 (왼쪽 패널)

연결이 선택되면 왼쪽 패널에 사용 가능한 모든 테이블과 뷰 목록이 표시됩니다.

- **통계**: 상단 요약에는 테이블과 뷰의 총 개수가 표시됩니다.
  - **필터**: "테이블" 또는 "뷰" 카드를 클릭하여 유형별로 목록을 필터링할 수 있습니다.
- **검색**: 이름으로 테이블을 필터링하는 실시간 검색 바.
- **목록 항목**:
  - **아이콘**: 테이블은 초록색, 뷰는 노란색 표시기로 구분됩니다.
  - **상태**: 현재 선택된 테이블은 배경이 강조 표시됩니다.

## 빈 상태 (Empty State)

연결이 선택되지 않은 경우, "플러그" 아이콘이 표시되며 탐색을 시작하려면 데이터베이스를 선택하라는 안내가 나옵니다.
